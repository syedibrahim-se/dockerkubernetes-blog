Red Hat Enterprise Linux 9 (RHEL9) and CentOS 9 are the latest versions of the popular Linux distributions. Follow Steps to Install Docer and Kubernetes:

Step-1: first we have to download and add docker repositry by using below command.

$ sudo dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

Step-2: After cloning repo we have to install the docker repo by using below command.

$ sudo dnf install docker-ce --nobest -y

Step-3: After that we want to start the docker engin by using a below command.

$ sudo systemctl start docker

Step-4: We want to enable the docker engine by using below command.

$ sudo systemctl enable docker

Step-5: Here check the docker engine working running or not by using below command.

$ sudo systemctl status docker

If showing active and running then it is successful.

Step-6: After successfully running docker engine.Now we can install kubernetes by downloading the kubernetes repo.

$ sudo curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"

Step-7: After downloading the repo we have to install kubernetes by using repo by using .

$ sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl

Step-8: Now here check the version of the kubernetes by using a below command.

$ kubectl version --output=yaml

Step-9: After successfully completed kubernetes installation then we have to install minikube by downloading the repo using below command.

$ sudo curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64

Step-10: Here i am changing the execute permission to downloade a repo file by using below command.

$ sudo chmod +x minikube-linux-amd64

Step-11: Moving the file home directory to usr/local directory using below command.

$ sudo mv minikube-linux-amd64 /usr/local/bin/minikube

Step-12: Starting the minikube using below command.

$ minikube start

Step-13: checking the kubernetes cluster info by using below command.

$ kubectl cluster-info

Step-14: To see the minikube dashboard info use below command.

$ minikube dashboard

Step-15: After seeing the minikube dashboard info we want to give access to the given port in our local command prompt by using below command.

$ ssh -L portnumber:localhost:portnumber instancename@ipaddress

Ex: ssh -L 42157:localhost:42157 aimaster@183.83.218.142

Step-16: After that open server terminal and copy the given dashboard link and past it on the web browser we can get kubernetes dashboard.

Ex: http://127.0.0.1:42157/api/v1/namespaces/kubernetes-dashboard/services/http:kubernetes-dashboard:/proxy/
